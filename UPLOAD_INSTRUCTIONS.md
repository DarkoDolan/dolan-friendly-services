# Upload instructions for GitHub Pages

## Create the site repo

1. Go to GitHub.
2. Click **+** → **New repository**.
3. Name it: `dolan-friendly-services`
4. Set it to **Public**.
5. Tick **Add a README file**.
6. Click **Create repository**.

## Upload this website kit

1. Open the extracted `DFS_Public_Website_v1_Kit` folder.
2. Go into the folder so you can see `index.html`, `README.md`, `assets`, `downloads`, and `copy`.
3. In GitHub, click **Add file** → **Upload files**.
4. Drag all of those items into GitHub.
5. Click **Commit changes**.

## Turn on Pages

1. In the repo, click **Settings**.
2. Click **Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main` and `/root`.
5. Click **Save**.

Your site should appear at:

`https://darkodolan.github.io/dolan-friendly-services/`

It can take a few minutes to publish.
